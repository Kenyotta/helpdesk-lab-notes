# Help Desk Lab Notes

This repository serves as my IT support and service desk portfolio. It contains documented ticket simulations, troubleshooting procedures, knowledge base articles, and lessons learned while developing practical experience in IT support and cybersecurity.

## Purpose

The goal of this repository is to build and demonstrate hands-on troubleshooting skills through service desk simulations and lab exercises. Each ticket is documented using a structured approach that includes problem identification, investigation, resolution, root cause analysis, and lessons learned.

## Skills Demonstrated

* Ticket Management
* Technical Troubleshooting
* Root Cause Analysis
* Documentation & Knowledge Base Creation
* Customer Support Workflows
* Windows Administration
* Active Directory Fundamentals
* Networking Fundamentals
* Cybersecurity Awareness

## Repository Structure

```text
tickets/
├── Documented ticket simulations

troubleshooting-guides/
├── Knowledge base articles
├── Step-by-step troubleshooting procedures

lessons-learned/
├── Weekly reflections
├── Skills gained and areas for improvement
```

## Objectives

* Develop practical IT support experience
* Improve troubleshooting methodology
* Build professional documentation skills
* Create a portfolio for Help Desk and Cybersecurity roles
* Track continuous learning and technical growth

## Disclaimer

All tickets and scenarios in this repository are based on lab environments, simulations, or educational exercises and do not contain real user or organizational data.
